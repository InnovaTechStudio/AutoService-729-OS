# Design System - AutoService

Este directorio contiene todos los assets y guías del sistema de diseño de AutoService.

## Contenido

- `/colors/`          → Paleta de colores oficial y variaciones
- `/typography/`      → Escala tipográfica, pesos y ejemplos
- `/components/`      → Componentes UI reutilizables (botones, tarjetas, etc.)
- `/icons/`           → Iconos personalizados (Lucide + modificaciones)
- `/spacing/`         → Guías de espaciado y layout
- `/assets/`          → Imágenes, logos, ilustraciones
- `style-guide.md`    → Documentación completa del Design System
- `Figma-link.txt`    → Enlace al archivo de Figma (si aplica)

## Propósito

Mantener consistencia visual entre:
- Landing Page
- Web Application (Panel de Control)
- Experiencia móvil

Todas las pantallas del proyecto deben seguir las reglas definidas aquí.
